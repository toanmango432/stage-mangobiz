## 1. Overview

The Clients (CRM) Module is the comprehensive client relationship management system. It handles client profiles, safety compliance (allergies, patch tests), loyalty programs, and client segmentation.

### Design Goals

| Goal | Description |
|------|-------------|
| **Speed** | Find any client in < 3 seconds |
| **Safety** | Allergy/alert info unmissable |
| **Completeness** | All client info in one view |
| **Compliance** | Form tracking with clear status |
| **Insight** | Key metrics visible at a glance |

## 2. Design Philosophy

### Core Principles

1. **Safety First** - Alerts and allergies always prominent
2. **Quick Access** - Most common actions in 1-2 clicks
3. **Progressive Detail** - Summary â†’ drill down
4. **Visual Status** - Colors communicate state
5. **Offline Resilient** - Sync status visible

### Visual Language

- **Clean Cards** - Information grouped logically
- **Status Badges** - Quick visual indicators
- **Alert Banners** - Unmissable safety info
- **Progress Indicators** - Loyalty tier progress

.client-row {
