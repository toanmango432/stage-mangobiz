## Responsive Behavior

### View Modes (All use same unified base)

**Compact (Line View)**
- Padding: 4-5px
- Border radius: 6px
- Height: ~28-32px
- 2-row layout

**Normal (Standard View)**
- Padding: 5-7px
- Border radius: 6px
- Height: ~50-60px
- 2-row responsive layout

**Grid Normal (Large Cards)**
- Padding: 11px
- Border radius: 8px
- Min height: 200-224px
- Flex column layout

**Grid Compact (Dense Grid)**
- Padding: 7-14px
- Border radius: 6px
- 4-row compact structure

**All views maintain:**
- Same base color
- Same texture
- Same shadow system
- Same accent strip position/width
- Same hover behavior
