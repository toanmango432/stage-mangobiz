## Animation Specifications

### Step Transition
```css
/* Content slides left when going forward */
.step-enter {
  transform: translateX(20px);
  opacity: 0;
}
.step-enter-active {
  transform: translateX(0);
  opacity: 1;
  transition: all 200ms ease-out;
}

/* Content slides right when going back */
.step-exit {
  transform: translateX(0);
  opacity: 1;
}
.step-exit-active {
  transform: translateX(-20px);
  opacity: 0;
  transition: all 200ms ease-out;
}
```

### Summary Panel Updates
```css
/* Price updates with subtle scale */
.price-update {
  animation: priceChange 300ms ease-out;
}

@keyframes priceChange {
  0% { transform: scale(1); }
  50% { transform: scale(1.05); color: teal-500; }
  100% { transform: scale(1); }
}
```
