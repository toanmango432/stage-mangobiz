## 1. Overview

The Team Module is a comprehensive staff management system delivered through the Admin Portal. It handles staff profiles, schedules, timesheets, payroll, and Mango's unique Turn Tracker feature.

### Design Goals

| Goal | Description |
|------|-------------|
| **Clarity** | Complex payroll data presented simply |
| **Efficiency** | Complete common tasks in < 3 clicks |
| **Accuracy** | Visual confirmation of all data entry |
| **Scalability** | Handle 50+ staff without performance issues |
| **Consistency** | Unified patterns across all team screens |

## 2. Design Philosophy

### Core Principles

1. **Data-Dense but Scannable** - Tables and lists with clear hierarchy
2. **Inline Editing** - Reduce modal fatigue for simple changes
3. **Progressive Complexity** - Basic settings visible, advanced collapsible
4. **Visual Feedback** - Status changes immediately reflected
5. **Offline Resilient** - Sync status always visible

### Visual Language

- **Professional & Clean** - Subtle colors, clear typography
- **Status-Driven** - Color-coded states throughout
- **Card-Based Layouts** - Grouped information in cards
- **Dense Tables** - Efficient data presentation

## 8. Accessibility

### 8.1 WCAG 2.1 AA Requirements

| Requirement | Specification |
|-------------|---------------|
| Color Contrast | 4.5:1 for text, 3:1 for UI |
| Focus Indicators | 2px solid outline, 2px offset |
| Screen Reader | Full ARIA labels |
| Keyboard Navigation | Tab order follows visual flow |
| Status Changes | Announce via aria-live |

### 8.2 Focus States

```css
:focus-visible {
  outline: 2px solid #2563EB;
  outline-offset: 2px;
}

tr:focus-visible {
  background-color: #EFF6FF;
}
```

### 8.3 Screen Reader Labels

| Element | Label |
|---------|-------|
| Staff row | "{name}, {role}, {status}, {service count} services" |
| Turn card | "{name}, {turn count} turns, {status}" |
| Adjust button | "Add turn for {name}" / "Remove turn from {name}" |
| Status badge | "Status: {status name}" |
| Tab | "{tab name}, {selected/not selected}" |

## 9. Animation Specifications

### 9.1 Durations

| Animation | Duration | Easing |
|-----------|----------|--------|
| Hover states | 150ms | ease-out |
| Panel slide | 300ms | ease-in-out |
| Tab switch | 200ms | ease |
| Turn update | 300ms | spring |
| Row expand | 250ms | ease-out |

### 9.2 Specific Animations

**Panel Slide-In:**
```css
@keyframes slide-in {
  from {
    transform: translateX(100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
```

**Turn Count Update:**
```css
@keyframes turn-pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.2); }
  100% { transform: scale(1); }
}
```

**Row Expand:**
```css
@keyframes expand {
  from {
    max-height: 56px;
    opacity: 0;
  }
  to {
    max-height: 300px;
    opacity: 1;
  }
}
```
