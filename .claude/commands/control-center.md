Navigate to the Control Center App folder and provide context about it.

The Control Center App is located at: `apps/control-center`

This is the admin/management dashboard for Mango. It includes:
- Multi-store management
- License management
- System configuration
- Analytics and reporting
- User/role management

To launch: `pnpm run dev` from the control-center directory.
