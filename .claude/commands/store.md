Navigate to the Store App folder and provide context about it.

The Store App is located at: `apps/store-app`

This is the main POS (Point of Sale) application for Mango. It includes:
- Front Desk management
- Booking system
- Checkout/Ticket panel
- Team management
- Client management

To launch it in Electron: `pnpm run electron:dev` from the store-app directory.
To launch in web mode: `pnpm run dev` from the store-app directory.
