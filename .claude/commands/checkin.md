Navigate to the Check-In App folder and provide context about it.

The Check-In App is located at: `apps/check-in`

This is the client-facing check-in kiosk application for Mango. It allows:
- Walk-in client self check-in
- Appointment check-in
- Queue management

To launch: `pnpm run dev` from the check-in directory.
